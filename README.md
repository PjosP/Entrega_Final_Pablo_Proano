# Entrega_Final_Pablo_Proano

Proyecto Final – Minería de Datos con Información de Aduana (SENAE)

Este repositorio contiene la versión final del proyecto de Minería de Datos. 
Aquí se encuentran todos los archivos del análisis, el modelado y los resultados obtenidos a partir de información real de importaciones del Servicio Nacional de Aduana del Ecuador (SENAE).

Estado del Proyecto
 - Proyecto finalizado.

Se realizaron todas las etapas previstas: 
 - carga de datos 
 - Limpieza
 - Normalización
 - Análisis exploratorio
 - Tratamiento de valores atípicos
 - Selección de variables
 - construcción de modelos de Random Forest.

Los resultados finales y conclusiones se encuentran documentados en el informe principal.


Entregables
 - Informe completo en RMarkdown (.Rmd).
 - Informe final en formato HTML (.html).
 - Informe final en formato PDF (.pdf).
 - Bases de datos utilizadas para el análisis.
 - Código completo y reproducible.

Notas Finales

 - Este repositorio contiene la versión final del trabajo para su revisión y evaluación.
